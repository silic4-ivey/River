To run:
1) Install JavaFX. Please see the following link on how to configure JavaFX with Eclipse https://stackoverflow.com/questions/33819052/how-do-i-import-javafx-into-eclipse
2) Compile and run the RiverServer program
3) Compile and run the RiverClient program multiple times to log in as multiple people.
4) Please note that firewall will have to be disabled for the program to work properly.
